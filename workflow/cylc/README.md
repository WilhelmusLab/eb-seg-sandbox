## OSCAR

The same Cylc configurations can be used on OSCAR, with the settings in `cylc/oscar/global.cylc`.

Install those using:
```bash
mkdir -p ~/.cylc/flow
cp ./oscar/global.cylc ~/.cylc/flow/global.cylc
```
